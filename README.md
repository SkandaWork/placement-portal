# Placement Portal Web Application

## Overview
This is a Flask-based web application for managing placement activities.

## Features
- Admin can approve companies
- Companies can create placement drives
- Students can apply to drives
- Application tracking system

## Technologies Used
- Python (Flask)
- SQLite
- HTML, CSS

## How to Run
1. Run: python app.py
2. Open: http://localhost:5000

## Default Admin Login
Username: admin  
Password: admin
